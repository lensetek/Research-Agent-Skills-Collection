Empowering AI with Scientific Rigor

Kumpulan skill terstruktur untuk **Research Agent** (plugin `science`). Mengotomatiskan seluruh alur penelitian mulai dari penemuan ide berbasis dataset, PRISMA SLR, pengolahan data statistik, hingga simulasi peer-review secara akurat dan bebas halusinasi.

Eksplorasi Skill Cara Pasang

Yang Baru / What's New

Pembaruan fitur dan kapabilitas terbaru untuk mendukung penelitian ilmiah yang kredibel.

Juli 2026

⚡ Optimisasi Arsitektur Hybrid Python + LLM NEW

Infrastruktur script terprogram berbasis Python CLI helper untuk memperkuat model lemah (seperti LLM 7B/8B lokal):

  • Zero-Hallucination Stats & Data: Uji statistik deskriptif dan komparasi dikalkulasi deterministik oleh script Python sebelum diproses LLM.
  • Workflow State Persistence: Fitur pelacakan fase progress.json dan modul json_repair.py menjamin eksekusi yang andal dan hemat biaya token.
Juli 2026

🛡️ Programmatic Citation Guardrail

Sistem verifikasi otomatis berbasis skrip Python untuk mengeliminasi halusinasi referensi:

  • Zero-Hallucination: Memvalidasi DOI dan kecocokan judul secara langsung menggunakan API Crossref dan OpenAlex secara deterministik.
  • Hard Gatekeeper: Menolak sitasi fiktif atau salah atribut (mismatch) secara otomatis sebelum diproses oleh model LLM.

Juli 2026

🗂️ Integrasi Obsidian Vault & Second Brain NEW

Integrasi ekspor otomatis luaran riset ke struktur catatan Obsidian Vault:

  • Vault Formatting: obsidian-vault-exporter (Skill ke-24) mentransformasikan literature notes, matriks SotA, dan hipotesis menjadi Markdown Obsidian-native.
  • Knowledge Graph: Dilengkapi YAML Frontmatter terstruktur, bidirectional WikiLinks ([[Links]]), serta template query Dataview visual.
Juli 2026

🌐 Integrasi Chrome DevTools MCP & Digital Twin

Integrasi browser otomatis untuk pengumpulan data dan simulasi interaksi kuesioner web:

  • Scraping Hibrida: data-acquisition-specialist mengutamakan API/HTTP cepat, dan beralih ke Chrome DevTools headless jika terblokir/dinamis.
  • Google Form Filler: synthetic-data-generator bertindak sebagai pengisi kuesioner otomatis dengan menyimulasikan profil persona digital twin.
Juli 2026

📊 Alur Dataset-Driven Ideation (Data Descriptors)

Mencari ide riset turunan dari publikasi Data Descriptor (Scientific Data, Data in Brief). Menawarkan dua jenis opsi:

  • Potensi 1: Ekstraksi rekomendasi riset masa depan langsung dari bagian Usage Notes.
  • Potensi 2: Analisis novelty dengan menghubungkan variabel baru menggunakan metode analisis/ML pilihan Anda.
Juni 2026

🇮🇩 Integrasi API Terbuka & Indeksasi Global (WoS, SINTA)

Pencarian literatur memprioritaskan API terbuka gratis (OpenAlex, arXiv, PubMed) dengan fallback otomatis tanpa konfigurasi API Key rumit. Dilengkapi audit indeksasi Web of Science (WoS) dan peringkat akreditasi jurnal nasional SINTA Indonesia (S1-S6).

Juni 2026

🔍 Upgrade PRISMA SLR & Skill Baru

Integrasi alur kerja PRISMA 4-tahap (Identification, Screening, Eligibility, Included) dengan filtering Python otomatis dan checkpoint CSV. Penambahan skill baru: model-evaluator-validator (Fase 2) dan journal-template-formatter (Fase 4).

Interactive Skills Explorer

Jelajahi 24 skill akademis yang terbagi ke dalam 5 fase pengerjaan riset terstruktur.

research-orchestrator

Pintu masuk utama untuk mengotomatiskan seluruh alur kerja riset 5 fase otomatis penuh, memanggil sub-skill secara beruntun, serta mendukung Implicit Personalization berbasis riwayat umpan balik pengguna.

Starter / Entry Point

discover-phenomenon-and-gap

Mengeksplorasi tren fenomena baru, menganalisis anomali literatur, memetakan celah ilmiah formal (Mode A), atau melakukan pencarian ide berbasis dataset sekunder (Mode B - Data Descriptors).

research-question-builder

Mengubah celah riset (gap) yang ditemukan menjadi rumusan masalah, Research Questions (RQ), tujuan penelitian, kontribusi ilmiah, dan batasan ruang lingkup riset secara formal.

hypothesis-or-proposition-builder

Menyusun pernyataan hipotesis (riset kuantitatif) atau proposisi (riset kualitatif) secara logis disertai visualisasi diagram Mermaid untuk kerangka konseptual teoretis.

research-design-planner

Merancang desain metode eksperimen (kuantitatif, kualitatif, mixed-method, AI/ML) lengkap dengan skenario validasi, pengujian robustness, dan rancangan ablation study.

synthetic-data-generator

Simulasi digital twin untuk membuat dataset sintetis secara offline dan mengotomatiskan pengisian kuesioner online (seperti Google Forms) via Chrome DevTools MCP.

data-acquisition-specialist

Mengumpulkan atau melakukan web scraping data menggunakan strategi hibrida (API/HTTP cepat dengan fallback ke browser headless Chrome DevTools MCP jika terblokir).

data-scientist-analyst

Melakukan pembersihan data, statistik deskriptif, analisis multivariat (regresi/PLS), pelatihan model ML, dan pembuatan visualisasi data riset secara otomatis.

model-evaluator-validator

Menguji signifikansi statistik (seperti T-Test, ANOVA, atau uji Diebold-Mariano) dari performa prediksi model untuk membuktikan keabsahan temuan riset.

literature-review-generator

Menyusun draf tinjauan pustaka terstruktur (Narrative, SLR, Thematic, Chronological, Critical) lengkap dengan alur PRISMA 4-tahap dan Argument Map.

extract-methodology

Mengekstrak 10 parameter teknis utama (tujuan, parameter eksperimen, dataset, dll.) dari teks penuh paper ilmiah secara akurat.

source-quality-appraiser

Mengaudit kredibilitas sumber rujukan dengan memetakan kuartil Scopus/WoS, peringkat konferensi CORE, pendeteksian bias, serta status akreditasi jurnal nasional SINTA (S1-S6).

citation-and-reference-validator

Audit keaslian rujukan (DOI) secara programatik via script Python terhadap API Crossref/OpenAlex serta memverifikasi kalimat sitasi di naskah untuk mengeliminasi halusinasi.

paper-matrix-builder

Mengotomatiskan pembuatan tabel ringkasan komparasi State-of-the-Art (SotA) dan peta bukti (evidence mapping) dari kumpulan paper rujukan.

synthesize-research

Menganalisis perbandingan temuan kritis lintas-paper, memetakan area konsensus atau kontradiksi ilmiah, dan menyaring opini agen.

patent-and-literature-matcher

Mengevaluasi kebaruan (novelty) dari ide invensi baru dengan membandingkan klaim teknologi terhadap potensi publikasi ilmiah prior art.

journal-recommendation-finder

Merekomendasikan target jurnal ilmiah yang cocok (Scopus, WoS, SINTA S1-S6) berdasarkan relevansi scope, kuartil, biaya publikasi (APC), dan kecepatan review.

academic-peer-reviewer

Simulasi peer review independen (Accept/Revision/Reject) dengan daftar saran perbaikan kritis (mayor & minor) sebelum disubmit ke jurnal sungguhan.

reviewer-response-and-revision

Membantu memformulasikan draf surat jawaban "Response to Reviewers" secara profesional serta menyusun rencana revisi naskah secara konsisten.

academic-paraphraser

Memparafrase draf teks akademik secara komparatif untuk menghindari plagiasi dan deteksi AI generator dengan tetap mempertahankan bobot ilmiah asli.

advanced-similarity-reducer

Penyusunan ulang struktur kalimat dan sintaksis tingkat lanjut untuk menurunkan persentase kemiripan (Turnitin) secara signifikan tanpa merusak konsep asli.

journal-template-formatter

Memformat ulang struktur, tata letak, dan gaya sitasi draf paper sesuai dengan Author Guidelines jurnal target (seperti IEEE, APA, Harvard, dll.).

obsidian-vault-exporter

Mentransformasikan seluruh dokumen luaran riset ke dalam berkas Markdown Obsidian-native lengkap dengan YAML Frontmatter terstruktur, WikiLinks 2 arah, tag, serta query Dataview visual.

agent-update-checker

Mengecek pembaruan (update) agent skills terbaru dari repositori GitHub secara berkala dan menawarkan pemasangan otomatis kepada pengguna.

user_profile.json

Berkas profil di root direktori yang mendasari Implicit Personalization. Menyimpan gaya penulisan, jurnal target, dan preferensi riset pengguna yang diperbarui secara otomatis.

Cara Pemasangan / Installation

Terdapat dua cara untuk memasang paket skill ini, melalui NPX atau langsung melalui obrolan dengan agen Anda.

Cara 1: NPX (Direkomendasikan)

Cara paling stabil untuk menginstal skill di workspace Anda.

Terminal / Bash
npx skills add https://github.com/lensetek/Research-Agent-Skills-Collection

Cara 2: Chat Prompt (Via Agen)

Pasang secara langsung dengan memberikan perintah/prompt kepada agen Anda.

Chat Prompt
Tolong download dan install agent skill dari url https://github.com/lensetek/Research-Agent-Skills-Collection

Langkah Verifikasi

Setelah proses pengunduhan selesai, tanyakan kepada agen Anda: "Konfirmasi berapa agent skill yang ada di project ini?" untuk memastikan 23 skill aktif.
